# Pykeys (WIP)

Python key and password generator tool.

## Requirements

Python 3.6+.

## Usage

### Key generation

To generate three keys with 256 characters:

```
pykeys 3 256
```

### Output

The output of the keys is printed to stdout and stored in a file called "pykeys.txt" in your 
current working directory.
